2. **验证 Git 配置**
   ```bash
   git config --global --list
   ```

3. **初始化 Git 仓库**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   ```

4. **连接到 GitHub 仓库**
   ```bash
   git remote add origin https://github.com/hi-znyan/hi-znyan.github.io.git
   ```

5. **推送到 GitHub**
   ```bash
   git push -u origin main
   ```

6. **配置 GitHub Pages**
   - 登录 GitHub，进入仓库 https://github.com/hi-znyan/hi-znyan.github.io
   - 点击 Settings > Pages
   - 选择 main 分支作为源
   - 点击 Save

7. **验证网站**
